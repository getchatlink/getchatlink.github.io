# 💬 GetChatLink - Generador de WhatsApp

Bienvenido al repositorio de **GetChatLink**, una herramienta web ligera y rápida para generar enlaces personalizados de WhatsApp y códigos QR descargables.

## 🚀 Características Principales

* **Generación de Enlaces:** Crea URLs de la API de WhatsApp (`wa.me`) listas para usar, con mensajes predeterminados codificados correctamente.
* **Limpieza de Datos:** Filtra automáticamente símbolos, guiones y espacios del número de teléfono ingresado.
* **Códigos QR Nativos:** Genera un código QR de la URL y lo renderiza como una imagen estática (`<img>`), permitiendo que los usuarios de dispositivos móviles puedan descargarlo simplemente manteniendo presionado el dedo sobre la pantalla.
* **Copy to Clipboard:** Botón de copiado rápido al portapapeles.

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Interfaz moderna, limpia y adaptada a dispositivos móviles (Mobile First).
* **JavaScript (Vanilla):** Manejo de eventos, sanitización de inputs y manipulación del DOM.
* **qrcode.js:** Librería ligera para la renderización matemática del código QR.
* **SEO:** Etiquetado semántico y seguimiento con Google Analytics.
